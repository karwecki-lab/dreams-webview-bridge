💡 PRZYKŁADY UŻYCIA

Wysyłanie z iOS do React:
  let message = ["type": "greeting", "payload": ["text": "Hi!"]]
  coordinator.sendMessageToJavaScript(message)

Wysyłanie z React do iOS:
  window.webkit.messageHandlers.nativeApp.postMessage({
    type: 'greeting',
    payload: { text: 'Hello!' }
  })

Dreams SDK:
  viewModel.logDreamsEvent(
    type: "user_action",
    data: ["action": "button_click"]
  )

🐛 TROUBLESHOOTING

Problem: "Cannot find package 'dreams-ios-sdk'"
  → File → Add Packages w Xcode
  → https://github.com/getdreams/dreams-ios-sdk

Problem: React pokazuje "Not in WKWebView"
  → To normalne w przeglądarce
  → Uruchom iOS app żeby zobaczyć pełną funkcjonalność

Problem: WebView pokazuje pustą stronę
  → Sprawdź czy React server działa (http://localhost:3000)
  → Sprawdź logi w Xcode

Więcej rozwiązań → README.md sekcja "Rozwiązywanie problemów"
